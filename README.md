# Cube-Game-MiniMax
---------------------------------------
Exercise at Univercity for AI lesson using MiniMax algorithm.
--------------------------------------------------------------
How to Play: 
The User choose a cube from the table and must get the last cube(1 or 2 or K) from the table to win.
